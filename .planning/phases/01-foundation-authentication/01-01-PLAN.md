| T-01-02 | Info Disclosure | `data/*.db` | mitigate | `.gitignore` adds `data/*`, `*.db`, `*.db-journal` patterns with `!data/.gitkeep` exception (Task 1 step 8) |
| T-01-03 | Tampering | `package.json` pinned versions | mitigate | Exact versions pinned per CLAUDE.md Version Compatibility table (Task 1 step 1); no `^` or `~` ranges that could pull unverified minor updates |
| T-01-04 | Info Disclosure | BETTER_AUTH_SECRET generation | mitigate | Generated via `crypto.randomBytes(32)` (Task 1 step 7) — 256 bits from OS CSPRNG, meets V6 cryptography ASVS |
</threat_model>
