<task id="05-02-01" type="auto">
<action>
Create `lib/history/queries.ts` with:
```ts
export async function listLauflistenHistoryForUser(userId: string, opts: {
  search?: string;
  dateFrom?: Date;
  dateTo?: Date;
  page?: number;
  pageSize?: number;
} = {}): Promise<{ items: HistoryRow[]; totalCount: number; page: number; pageSize: number }>
```

Uses Drizzle query: JOIN `laufliste` with `case` on `caseId`, WHERE `case.userId = userId` + optional LIKE on `case.personName` + optional generatedAt range. ORDER BY `generatedAt DESC`. Pagination via LIMIT/OFFSET.

HistoryRow = { lauflisteId, caseId, personName, documentCount, fileSize, generatedAt }.

Also create `lib/history/queries.test.ts` with 3 cases: search match, date range, pagination+total.

Commit: `feat(05-02): add lauflisten history queries`.
</action>
