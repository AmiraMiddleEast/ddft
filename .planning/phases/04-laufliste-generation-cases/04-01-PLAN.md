  <action>
Append three new Drizzle tables to `db/schema.ts` following the conventions already established in Phases 1-3 (see <interfaces>). Do NOT modify existing tables.

1. Export `CASE_STATUS = ["open","ready_for_pdf","pdf_generated"] as const` and `type CaseStatus`.

2. Export `caseTable` (table name string: `"case"`) with columns per CONTEXT D-01:
   - `id: text("id").primaryKey()`
   - `userId: text("user_id").notNull().references(() => user.id, { onDelete: "cascade" })`
   - `personName: text("person_name").notNull()`
   - `personBirthdate: text("person_birthdate")` (nullable; ISO yyyy-MM-dd stored as text)
   - `notes: text("notes")` (nullable)
   - `status: text("status", { enum: CASE_STATUS }).notNull().default("open")`
   - `createdAt` + `updatedAt` using the subsecond timestamp pattern (see <interfaces>); `updatedAt` uses `.$onUpdate(() => new Date())`.
   - Indexes: `index("case_user_idx").on(t.userId)`, `index("case_user_updated_idx").on(t.userId, t.updatedAt)`.
   - Check constraint: `check("case_status_ck", sql`${t.status} IN ('open','ready_for_pdf','pdf_generated')`)`.

3. Export `caseDocument` (table name: `"case_document"`):
   - `id: text("id").primaryKey()`
   - `caseId: text("case_id").notNull().references(() => caseTable.id, { onDelete: "cascade" })`
   - `documentId: text("document_id").notNull().references(() => document.id, { onDelete: "cascade" })`
   - `position: integer("position").notNull()`
   - `addedAt` timestamp (same subsecond pattern; no $onUpdate).
   - Indexes per D-02: `uniqueIndex("case_document_doc_uniq").on(t.documentId)` (global one-case-per-doc), `uniqueIndex("case_document_case_doc_uniq").on(t.caseId, t.documentId)`, `index("case_document_case_pos_idx").on(t.caseId, t.position)`.

4. Export `laufliste` (table name: `"laufliste"`):
   - `id: text("id").primaryKey()`
   - `caseId: text("case_id").notNull().references(() => caseTable.id, { onDelete: "cascade" })`
   - `userId: text("user_id").notNull().references(() => user.id, { onDelete: "cascade" })`
   - `pdfStoragePath: text("pdf_storage_path").notNull()`
   - `generatedAt` timestamp (subsecond default; no $onUpdate).
   - `documentCount: integer("document_count").notNull()`
   - `fileSize: integer("file_size").notNull()`
   - Indexes: `index("laufliste_case_idx").on(t.caseId)`, `index("laufliste_case_generated_idx").on(t.caseId, t.generatedAt)`.

Import `check`, `uniqueIndex`, `index`, `sql` if not already imported. Place new tables below existing Phase 3 additions, with a `// ----- Phase 4 ----- ` header comment. Do not add relations() blocks — queries will join manually.

Implements D-01, D-02.
